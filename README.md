# Customization

All colors are controlled via CSS variables inside `:root`:

```css
:root {
    --base: #1A0F14;
    --mantle: #22141A;
    --crust: #2B1A21;

    --maroon: #6E1F2B;
    --red: #8C2F39;
    --flamingo: #E8A8AF;
}
```
